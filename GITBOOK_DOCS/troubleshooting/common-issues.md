# Common Issues

This section covers the most frequently encountered issues and their solutions.

---

## Installation Issues

### Extension Not Loading
**Problem**: AutoNestCut doesn't appear in Extensions menu after installation

- Restart SketchUp completely
- Check Window â†’ Extension Manager for installation status
- Verify RBZ file integrity by re-downloading

### Ruby Console Errors
**Problem**: Error messages appear in Ruby Console during startup

**Solutions**:
- Check for conflicting extensions
- Ensure SketchUp version compatibility (2020+)
- Contact support with specific error messages

## Usage Issues

### No Components Detected
**Problem**: "No valid components found" message appears

**Solutions**:
- Ensure components are selected before running analysis
- Check that selected objects are components or groups
- Verify components have valid dimensions
- Make sure materials are properly assigned

### Material Recognition Problems
**Problem**: Materials not detected or incorrectly categorized

**Solutions**:
- Apply materials directly to component faces
- Use consistent material naming conventions
- Check material properties in SketchUp Material Browser
- Ensure materials represent sheet goods

### Nesting Optimization Issues
**Problem**: Poor nesting results or excessive waste

**Solutions**:
- Verify component dimensions are accurate
- Check kerf settings match your cutting method
- Try different optimization algorithms
- Ensure sheet sizes are appropriate for parts

## Performance Issues

### Slow Analysis
**Problem**: Cut list generation takes excessive time

**Solutions**:
- Reduce number of selected components
- Simplify complex nested component structures
- Close unnecessary SketchUp models
- Restart SketchUp to clear memory

### Memory Errors
**Problem**: Out of memory errors during processing

**Solutions**:
- Process smaller batches of components
- Close other applications to free memory
- Restart SketchUp between large analyses
- Consider upgrading system RAM

## Licensing Issues

### Trial Limitations
**Problem**: Features disabled or trial expired

**Solutions**:
- Purchase full license for unlimited access
- Contact support for license assistance
- Verify internet connection for license validation

### License Activation Problems
**Problem**: Valid license not recognized

**Solutions**:
- Check internet connection
- Verify license key accuracy
- Contact support with license details
- Try restarting SketchUp
