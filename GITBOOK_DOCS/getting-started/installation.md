# Installation

This guide covers the installation process for AutoNestCut extension in SketchUp.

---

## Prerequisites

Before installing AutoNestCut, ensure your system meets the following requirements:

*   SketchUp 2020 or later
*   Windows operating system
*   Minimum 4GB RAM
*   Administrative privileges for installation

## Installation Steps

1.  **Download Extension**
    *   Download the AutoNestCut.rbz file from your purchase confirmation email
    *   Save the RBZ file to your Downloads folder or desktop

2.  **Install via SketchUp Extension Manager**
    *   Open SketchUp
    *   Navigate to Window → Extension Manager
    *   Click "Install Extension" button
    *   Browse and select the AutoNestCut.rbz file
    *   Click "Yes" to confirm installation

3.  **Verify Installation**
    *   Check Extensions menu for "AutoNestCut" submenu
    *   Verify the AutoNestCut toolbar appears
    *   Extension is ready to use immediately

## License Activation

> [!NOTE]
> AutoNestCut includes a 7-day free trial. No license key is required for initial evaluation.

After installation, you can:

*   Use the extension immediately with trial limitations
*   Purchase a license when ready for full functionality
*   Enter your license key via Extensions → AutoNestCut → License Info

## Troubleshooting Installation

**Extension Not Appearing**
*   Restart SketchUp completely after installation
*   Check Window → Extension Manager to verify installation
*   Ensure the RBZ file downloaded completely

**Installation Errors**
*   Run SketchUp as administrator
*   Temporarily disable antivirus software if needed
*   Try downloading the RBZ file again

**Ruby Console Errors**
*   Open Window → Ruby Console to view error messages
*   Contact support with specific error details if issues persist
