# Features Overview

AutoNestCut provides comprehensive tools for optimizing sheet goods usage in woodworking and manufacturing projects.

---

## Core Features

### Automated Cut List Generation
- Analyzes selected SketchUp components automatically
- Extracts dimensions and material information
- Groups parts by material type and thickness
- Generates detailed cut lists with quantities

### Optimized Nesting
- Advanced algorithms minimize material waste
- Multiple optimization strategies available
- Visual nesting diagrams show optimal layouts
- Supports various sheet sizes and orientations

### Material Analysis
- Intelligent material detection from SketchUp models
- Categorizes components by material properties
- Filters sheet goods from other component types
- Handles complex nested component structures

### Visual Reports
- Professional cut list reports
- Nesting diagram exports
- Material usage summaries
- Waste analysis and optimization suggestions

## Advanced Capabilities

### Component Analysis
- Recursive analysis through all nesting levels
- Hierarchical component tree visualization
- Handles groups and component instances
- Maintains material inheritance

### Export Options
- PDF reports for workshop use
- Image exports for documentation
- Detailed material breakdowns
- Cost estimation support

### Optimization Settings
- Configurable kerf allowances
- Multiple sheet size options
- Grain direction considerations
- Custom optimization parameters
