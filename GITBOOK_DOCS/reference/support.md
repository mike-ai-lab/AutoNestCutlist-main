## What to Include in Support Requests

**Technical Issues**
*   SketchUp version and operating system
*   AutoNestCut version number
*   Detailed description of the problem
*   Steps to reproduce the issue
*   Screenshots or screen recordings if applicable
*   Sample SketchUp model files (if relevant)

**License Issues**
*   License type (trial, annual, lifetime)
*   Purchase confirmation details
*   Error messages during activation
*   Hardware configuration changes

## Bug Reports

**Reporting Process**
1.  Check existing documentation for known issues
2.  Reproduce the problem consistently
3.  Gather system information and error details
4.  Send detailed report to support email
5.  Await confirmation and tracking information

**Bug Report Template**
```
Subject: AutoNestCut Bug Report - [Brief Description]

System Information:

- AutoNestCut Version:
```
