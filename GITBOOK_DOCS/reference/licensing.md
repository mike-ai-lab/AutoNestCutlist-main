## License Activation

**Automatic Trial**
*   Trial begins on first extension use
*   No registration required
*   Countdown displayed in extension interface

**License Key Entry**
1.  Navigate to Extensions → AutoNestCut → License Info
2.  Click "Enter License Key" if prompted
3.  Paste your license key from purchase email
4.  Click "Activate License"
5.  Restart SketchUp to complete activation

**Offline Activation**
*   Contact support for offline activation assistance
*   Provide workstation hardware fingerprint
*   Receive offline activation file via email

## License Management

**Viewing License Status**
*   Extensions → AutoNestCut → License Info
*   Displays current license type and expiration
*   Shows remaining trial days if applicable

**License Transfer**
*   Remove license from current workstation
*   Install and activate on new workstation
*   One active installation per license

**License Renewal**
*   Annual licenses require yearly renewal
*   Renewal notices sent via email before expiration
*   Lifetime licenses never expire
