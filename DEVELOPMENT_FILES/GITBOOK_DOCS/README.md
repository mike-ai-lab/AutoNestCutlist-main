# AutoNestCut

Professional SketchUp extension for automated material optimization and cut list generation.

---

## Overview

AutoNestCut is a comprehensive SketchUp extension designed for woodworkers, furniture makers, and architects who need to optimize material usage and generate cutting diagrams for sheet goods. The extension automatically analyzes 3D model components and creates efficient nesting layouts to minimize waste and reduce project costs.

> [!NOTE]
> AutoNestCut requires SketchUp 2020 or later and includes a 7-day free trial for evaluation.

## Key Features

*   **Automatic Part Detection** - Intelligently identifies sheet good components based on dimensions and thickness
*   **Material Management** - Configure stock materials with custom dimensions, pricing, and currency
*   **Multi-Currency Support** - 100+ international currencies with automatic symbol formatting
*   **Internationalization** - Available in English, Spanish, French, and German
*   **Optimized Nesting** - Advanced algorithms minimize waste by efficiently arranging parts
*   **Visual Diagrams** - Generate clear cutting diagrams with part labels and dimensions
*   **Enhanced UI** - Professional typography with improved readability and accessibility
*   **Detailed Reports** - Comprehensive reports with material lists, costs, and efficiency metrics
*   **Multiple Export Formats** - PDF, CSV, and interactive HTML exports

## Quick Start

1.  **Select Components** - Choose the components or groups you want to analyze in your SketchUp model
2.  **Launch Extension** - Navigate to Extensions → AutoNestCut → Generate Cut List
3.  **Configure Settings** - Set kerf width, material properties, and optimization preferences
4.  **Generate Report** - Process the nesting optimization and review cutting diagrams
5.  **Export Results** - Save results as PDF for printing, CSV for spreadsheets, or HTML for sharing
