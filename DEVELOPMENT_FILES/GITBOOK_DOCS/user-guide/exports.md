# Export Options

## Export Formats

**PDF Export**
*   Professional documentation format
*   Print-ready cutting diagrams
*   Complete project reports with tables
*   Scalable vector graphics for clarity

**CSV Export**

**HTML Export**
*   Interactive web-based reports
*   3D part visualization capabilities
*   Responsive design for mobile devices
*   Self-contained files for easy sharing
