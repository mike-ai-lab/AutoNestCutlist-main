# System Requirements

Technical specifications and compatibility information for AutoNestCut.

---

## Minimum Requirements

**SketchUp Version**
*   SketchUp 2020 or later
*   SketchUp Pro, Make, or Web (desktop) editions supported
*   Both subscription and perpetual licenses compatible

**Operating System**
*   Windows 10 or later
*   Windows Server 2016 or later
*   64-bit architecture required

**Hardware Specifications**
*   4GB RAM minimum (8GB recommended)
*   1GB available disk space
*   Graphics card supporting OpenGL 3.0 or later
*   Internet connection for license activation

## Recommended Configuration

**For Optimal Performance**
*   8GB RAM or more
*   SSD storage for faster file operations
*   Multi-core processor (Intel i5 or AMD equivalent)
*   Dedicated graphics card

**For Large Projects**
*   16GB RAM for complex models with 500+ components
*   High-resolution display (1920Ã—1080 minimum)
*   Multiple monitors for enhanced workflow

## Compatibility Notes

**SketchUp Extensions**
*   Compatible with most popular SketchUp extensions
*   No known conflicts with major plugins
*   Ruby 2.7+ environment required

**File Formats**
*   Works with standard SketchUp (.skp) files
*   Supports SketchUp components and groups
*   Compatible with imported geometry from other CAD systems

**Network Environments**
*   Functions in corporate network environments
*   Proxy server compatibility for license validation
*   Offline operation after initial license activation

## Performance Considerations

**Model Complexity**
*   Optimized for models with up to 1000 components
*   Performance scales with available system memory
*   Processing time increases with part count and complexity

**Memory Usage**
*   Typical usage: 100-500MB additional RAM
*   Large projects may require up to 2GB
*   Automatic memory cleanup between sessions

## Licensing Requirements

**Trial Version**
*   7-day evaluation period
*   Full feature access during trial
*   No credit card required for trial activation

**Licensed Version**
*   Internet connection required for initial activation
*   Offline operation after activation
*   Single workstation license per purchase
