# Development Files

This folder contains development files, temporary scripts, documentation, and backup copies that are NOT part of the final extension distribution.

## Contents

- **AutoNestCut_Clean_Workspace_BACKUP/**: Complete backup of the working extension before encryption
- **GITBOOK_DOCS/**: Documentation source files for GitBook
- **Server/**: Standalone server application (separate from extension)
- **Development Files**: Various temporary files, test outputs, and development artifacts

## Important Notes

- These files are for development reference only
- The actual extension files are in the root directory
- Server folder is a separate application, not part of the SketchUp extension
- Backup folder preserves the complete working state before security packaging

## Do Not Include

These files should NOT be included in the final RBZ extension package:
- Documentation source files
- Server application
- Test outputs and temporary files
- Development scripts and loaders
- Backup copies
