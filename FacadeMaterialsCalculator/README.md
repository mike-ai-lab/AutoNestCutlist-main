# Facade Materials Calculator

A standalone SketchUp extension for calculating facade cladding materials.

## Features

- Analyzes selected facade surfaces
- Calculates material quantities based on piece dimensions
- Supports common materials (brick, stone, tiles)
- Includes waste factor calculations
- Cost estimation
- Clean, professional UI

## Installation

1. Load the extension in SketchUp Ruby Console:
```ruby
load "C:/path/to/FacadeMaterialsCalculator/facade_calculator.rb"
```

2. Or copy to SketchUp Plugins folder

## Usage

1. Select facade surfaces (faces) in SketchUp
2. Go to Extensions > Facade Materials Calculator
3. Choose material preset or enter custom dimensions
4. Set waste factor and costs
5. Click Calculate Materials

## Independent Extension

This is completely independent from AutoNestCut and can be used standalone.
