# AutoNestCut - SketchUp Extension

AutoNestCut is a SketchUp extension that generates optimized cut lists and nesting diagrams for sheet goods, helping woodworkers and manufacturers minimize material waste.

## Purpose

- **Automated Cut List Generation**: Analyzes selected components and generates detailed cut lists
- **Optimized Nesting**: Creates efficient layouts to minimize sheet material waste
- **Material Analysis**: Identifies and categorizes sheet goods from your SketchUp model
- **Export Reports**: Generates visual diagrams and detailed reports

## Requirements

- SketchUp 2020 or later
- Valid license (trial available)

## Usage

### Basic Workflow

1. **Select Components**: Select the components or groups you want to analyze
2. **Generate Cut List**: 
   - Use toolbar button, or
   - Go to Extensions → AutoNestCut → Generate Cut List
3. **Configure Settings**: Choose materials, board sizes, and optimization preferences
4. **Export Results**: Generate nesting diagrams and cut list reports

### Menu Options

- **Generate Cut List**: Main function to analyze selected components
- **Documentation**: Built-in help and usage guide
- **License Info**: View license status and purchase options
- **Trial Status**: Check remaining trial time

## Features

- Recursive component analysis through all nesting levels
- Material-based part categorization
- Hierarchical component tree visualization
- Sheet goods detection and filtering
- Optimized nesting algorithms
- Visual diagram generation
- Detailed cut list reports

## Installation

1. Copy the extension folder to your SketchUp Plugins directory
2. Restart SketchUp
3. The AutoNestCut toolbar and menu will appear automatically
